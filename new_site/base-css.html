Typography

Headings

All HTML headings, <h1> through <h5> are available.

<h1> h1. Heading 1

<h2> h2. Heading 2

<h3> h3. Heading 3

<h4> h4. Heading 4

<h5> h5. Heading 5
<h1>...</h1>

Headings options

Create variants of the headings using .white and .feature. The .white class can be applied to change the text color to white and it can be used for all headings. The .feature class can only be used for the <h4> heading.

<h4 class="feature"> h4. Heading 4

<h1 class="white-text"> h1. Heading 1

<h4 class="feature">...</h4>
<h1 class="white-text" >...</h1>

Body copy

Sherpa's global default font-size is 14px, with a line-height of 20px. This is applied to the <body> and all paragraphs. In addition, <p> (paragraphs) receive a bottom margin of half their line-height (10px by default).

Emphasis

Make use of HTML's default emphasis tags with lightweight styles.

Small

<small>

For de-emphasizing inline or blocks of text, use the small tag.

This line of text is meant to be treated as fine print.

<p><small>This line of text is meant to be treated as fine print.</small></p>

Bold

For emphasizing a snippet of text with a heavier font-weight.

The following snippet of text is rendered as bold text.

<strong>rendered as bold text</strong>

Italics

For emphasizing a snippet of text with italics.

The following snippet of text is rendered as italicized text.

<em>rendered as italicized text</em>

Tables

Default styles

For basic styling—light padding and only horizontal dividers—add the base class .table to any <table>.

# First Name Last Name Username
1 John Smith bigboy
2 Sara Longlastname slonglastname
3 Jim Barker barkerboy 
<table class="table">
  …
</table>

Optional classes

Add any of the following classes to the .table base class.

.table-striped

Adds zebra-striping to any table row within the <tbody> via the :nth-child CSS selector (not available in IE7-8).

# First Name Last Name Username
1 John Smith bigboy
2 Sara Longlastname slonglastname
3 Jim Barker barkerboy 
<table class="table table-striped">
  …
</table>

.table-bordered

Add borders and rounded corners to the table.

# First Name Last Name Username
1 John Smith bigboy
2 Sara Longlastname slonglastname
3 Jim Barker barkerboy 
<table class="table table-bordered">
  …
</table>

.table-hover

Enable a hover state on table rows within a <tbody>.

# First Name Last Name Username
1 John Smith bigboy
2 Sara Longlastname slonglastname
3 Jim Barker barkerboy 
<table class="table table-hover">
  …
</table>

.table-condensed

Makes tables more compact by cutting cell padding in half.

# First Name Last Name Username
1 John Smith bigboy
2 Sara Longlastname slonglastname
3 Jim Barker barkerboy 
<table class="table table-condensed">
  …
</table>

Optional row classes

Use contextual classes to color table rows.

Class Description
.success Indicates a successful or positive action.
.error Indicates a dangerous or potentially negative action.
.warning Indicates a warning that might need attention.
.info Used as an alternative to the default styles.

4 TB - Monthly 04/04/2012 Call in to confirm

Supported table markup

List of supported table HTML elements and how they should be used.

Tag Description
<table> Wrapping element for displaying data in a tabular format
<thead> Container element for table header rows (<tr>) to label table columns
<tbody> Container element for table rows (<tr>) in the body of the table
<tr> Container element for a set of table cells (<td> or <th>) that appears on a single row
<td> Default table cell
<th> Special table cell for column (or row, depending on scope and placement) labels
<caption> Description or summary of what the table holds, especially useful for screen readers

  7.     </tr>
  8.   </thead>
  9.   <tbody>

Forms

Text Fields

A one-line input field that a user can enter text into.

With Label

<label>First Name</label>
<input type="text">

Without Label

<input type="text" placeholder="First Name">

Radio Buttons

Radio buttons let a user select ONLY ONE of a limited number of choices.

Stacked

<label class="radio">
  <input type="radio" name="optionsRadio" value="Option one" checked> Option one
</label>
<label class="radio">
  <input type="radio" name="optionsRadio" value="Option two"> Option two
</label>

Inline

Add the .inline class to display a series of radios on one line.

<label class="radio inline">
  <input type="radio" name="optionsRadio" value="Option one" checked> Option one
</label>
<label class="radio inline">
  <input type="radio" name="optionsRadio" value="Option two"> Option two
</label>

Checkboxes

Checkboxes enable the user to make multiple selections.

Stacked

<label class="checkbox">
  <input type="checkbox" name="optionsCheckbox" value="Option one" checked> Option one
</label>
<label class="checkbox">
  <input type="checkbox" name="optionsCheckbox" value="Option two"> Option two
</label>

Inline

Add the .inline class to display a series of checkboxes on one line.

<label class="checkbox inline">
  <input type="checkbox" name="optionsCheckbox" value="Option one" checked> Option one
</label>
<label class="checkbox inline">
  <input type="checkbox" name="optionsCheckbox" value="Option two"> Option two
</label>

Select Lists

Select Lists allow the user to select one option from a list of pre-defined options.

With Label

<label>State</label>
<select>
  <option>1</option>
  <option>2</option>
  <option>3</option>
  <option>4</option>
  <option>5</option>
</select>

Without Label

<select>
  <option>State</option>
  <option>1</option>
  <option>2</option>
  <option>3</option>
  <option>4</option>
</select>

Search Form

A one-line input field with an appended button. Wrap the input and <button> inside <div class="input-append">.

<div class="input-append">
  <input type="search">
  <button type="button" class="btn btn-tertiary">
  <span aria-hidden="true" class="icon-search-right"></span>
  </button>
</div>

Prepended Input

A one-line text field with pre-populated content before the input. Wrap the <span class="add-on"> and input inside <div class="input-prepend">.

$ 
<div class="input-prepend">
  <span class="add-on">$</span>
  <input type="text" placeholder="Username">
</div>

Help Text

Block and Inline support for help and info text that appears within forms.

Block Help

To display Block Help Text add a <span class="help-block"> after input.

This is block help text 
<input type="text" placeholder="Username">
<span class="help-block">This is block help text</span>

Inline Help

To display Inline Help Text add a <span class="help-inline"> after input.

Note Inline Help text must use a relative sized input class.

Inline help text 
<input class="input-large" type="text" placeholder=".input-large">
<span class="help-inline">
  <span aria-hidden="true" class="icon-finance"> Inline help text</span>
</span>

Disabled

Add the disabled attribute on an input, radio button or checkbox to prevent user input and trigger a disabled display.

<input type="text" placeholder="Disabled Input" disabled>
<label class="checkbox">
  <input type="checkbox" name="optionsCheckbox" value="Option one" disabled> Disabled Checkbox
</label>
<label class="radio">
  <input type="radio" name="optionsRadio" value="Option one" disabled> Disabled Radio Button
</label>

Default Layouts

Individual form controls receive styling, but without any required base class on the <form> and <fieldset> layouts will be affected.

The <form> tag is used to create an HTML form for user input. Any form element must be within the <form> tag.

The <fieldset> tag is used to group related elements in a form. Sherpa adds a 10px margin to the bottom of a <fieldset>

Forgot Password

      <input type="text" placeholder="First Name">
    </div>    
    <div class="da4-da7">
      <input type="text" placeholder="Last Name">
    </div>  
    <div class="da4-da7">
      <p class="blue-text">Forgot Password</p>
    </div> 
    <div class="da-all">
      <button type="submit" class="btn btn-primary"<Submit</button<
      <button type="submit" class="btn btn-tertiary"<Create an Account</button<
    </div>
  </fieldset>
</form>
