Typography

Headings

All HTML headings, <h1> through <h5> are available.

<h1> h1. Heading 1

<h2> h2. Heading 2

<h3> h3. Heading 3

<h4> h4. Heading 4

<h5> h5. Heading 5
<h1>...</h1>

Headings options

Create variants of the headings using .title .feature and .white classes.

Page Title

The Page Title is a <h1> contained within a <div class="title">

Page Title

<div class="title">
  <h1>Page Title</h1>
<div>

Feature Heading

The .feature class can only be used for the <h4> heading.

Feature Heading

<h4 class="feature">Feature Heading</h4>

White Heading

Create a White Heading using the .white class. The .white class can can be used for all headings.

White Heading

Lorem ipsum dolor sit amet Consectetur adipiscing elit Integer molestie lorem at massa Facilisis in pretium nisl aliquet Nulla volutpat aliquam velit

<h1 class="white-text">White Heading</h1>

Body copy

Sherpa's global default font-size is 14px, with a line-height of 20px. This is applied to the <body> and all paragraphs. In addition, <p> (paragraphs) receive a bottom margin of half their line-height (10px by default).

Nullam quis risus eget urna mollis ornare vel eu leo. Cum sociis natoque penatibus et magnis dis parturient montes, nascetur ridiculus mus. Nullam id dolor id nibh ultricies vehicula.

Cum sociis natoque penatibus et magnis dis parturient montes, nascetur ridiculus mus. Donec ullamcorper nulla non metus auctor fringilla. Duis mollis, est non commodo luctus, nisi erat porttitor ligula, eget lacinia odio sem nec elit. Donec ullamcorper nulla non metus auctor fringilla.

Maecenas sed diam eget risus varius blandit sit amet non magna. Donec id elit non mi porta gravida at eget metus. Duis mollis, est non commodo luctus, nisi erat porttitor ligula, eget lacinia odio sem nec elit.

Emphasis

Make use of HTML's default emphasis tags with lightweight styles.

Small

<small>

For de-emphasizing inline or blocks of text, use the small tag.

This line of text is meant to be treated as fine print.

<p><small>This line of text is meant to be treated as fine print.</small></p>

Bold

For emphasizing a snippet of text with a heavier font-weight.

The following snippet of text is rendered as bold text.

<strong>rendered as bold text</strong>

Italics

For emphasizing a snippet of text with italics.

The following snippet of text is rendered as italicized text.

<em>rendered as italicized text</em>

Tables

Default

For basic styling—light padding and only horizontal dividers—add the base class .table to any <table>.

# First Name Last Name Username
1 John Smith bigboy
2 Sara Longlastname slonglastname
3 Jim Barker barkerboy 
<table class="table">
  …
</table>

Optional Tables

Add any of the following classes to the .table base class.

Striped

.table-striped adds zebra-striping to any table row within the <tbody> via the :nth-child CSS selector (not available in IE7-8).

# First Name Last Name Username
1 John Smith bigboy
2 Sara Longlastname slonglastname
3 Jim Barker barkerboy 
<table class="table table-striped">
  …
</table>

Bordered

.table-bordered adds borders and rounded corners to the table.

# First Name Last Name Username
1 John Smith bigboy
2 Sara Longlastname slonglastname
3 Jim Barker barkerboy 
<table class="table table-bordered">
  …
</table>

Hover

.table-hover enables a hover state on table rows within a <tbody>.

# First Name Last Name Username
1 John Smith bigboy
2 Sara Longlastname slonglastname
3 Jim Barker barkerboy 
<table class="table table-hover">
  …
</table>

Condensed

.table-condensed makes tables more compact by cutting cell padding in half.

# First Name Last Name Username
1 John Smith bigboy
2 Sara Longlastname slonglastname
3 Jim Barker barkerboy 
<table class="table table-condensed">
  …
</table>

Optional row classes

Use contextual classes to color table rows.

Class Description

.info Used as an alternative to the default styles.

Supported table markup

List of supported table HTML elements and how they should be used.

Tag Description
<table> Wrapping element for displaying data in a tabular format
<thead> Container element for table header rows (<tr>) to label table columns
<tbody> Container element for table rows (<tr>) in the body of the table
<tr> Container element for a set of table cells (<td> or <th>) that appears on a single row
<td> Default table cell
<th> Special table cell for column (or row, depending on scope and placement) labels
<caption> Description or summary of what the table holds, especially useful for screen readers 
<table>
  <caption>...</caption>
  <thead>
    <tr>
      <th>...</th>
      <th>...</th>
    </tr>
  </thead>
    <tbody>
      <tr>
        <td>...</td>
        <td>...</td>
      </tr>
    </tbody>
</table>

Forms

Text Fields

A one-line input field that a user can enter text into.

With Label

<label>First Name</label>
<input type="text">

Without Label

<input type="text" placeholder="First Name">

Radio Buttons

Radio buttons let a user select ONLY ONE of a limited number of choices.

Stacked

<label class="radio">
  <input type="radio" name="optionsRadio" value="Option one" checked> Option one
</label>
<label class="radio">
  <input type="radio" name="optionsRadio" value="Option two"> Option two
</label>

Inline

Add the .inline class to display a series of radios on one line.

<label class="radio inline">
  <input type="radio" name="optionsRadio" value="Option one" checked> Option one
</label>
<label class="radio inline">
  <input type="radio" name="optionsRadio" value="Option two"> Option two
</label>

Stacked

Search Form

A one-line input field with an appended button. Wrap the input and <button> inside <div class="input-append">.

<div class="input-append">
  <input type="search">
  <button type="button" class="btn btn-tertiary">
  <span aria-hidden="true" class="icon-small-search"></span>
  </button>
</div>

Prepended Input

A one-line text field with pre-populated content before the input. Wrap the <span class="add-on"> and input inside <div class="input-prepend">.

$ 
<div class="input-prepend">
  <span class="add-on">$</span>
  <input type="text" placeholder="Username">
</div>

Help Text

Block and Inline support for help and info text that appears within forms.

Block Help

To display Block Help Text add a <span class="help-block"> after input.

This is block help text 
<input type="text" placeholder="Username">
<span class="help-block">This is block help text</span>

Inline Help

To display Inline Help Text add a <span class="help-inline"> after input.

Note Inline Help text must use a relative sized input class.

Inline help text 
<input class="input-large" type="text" placeholder=".input-large">
<span class="help-inline">
  <span aria-hidden="true" class="icon-finance"> Inline help text</span>
</span>

Add the disabled attribute on an input, radio button or checkbox to prevent user input and trigger a disabled display.

<input type="text" placeholder="Disabled Input" disabled>

</label>
<label class="radio">
  <input type="radio" name="optionsRadio" value="Option one" disabled> Disabled Radio Button
</label>

Buttons

Default

Button styles can be applied to anything with the .btn class applied. However, typically you'll want to apply these to only <a> and <button> elements for the best rendering.

Button class="" Description
btn Standard gray button with gradient
btn btn-primary Used for purchase path, guiding a customer toward checkout.
btn btn-secondary Used as a call-to-action that does not take the user down the purchase path.
btn btn-tertiary Used for additional button needs.
btn btn-link Deemphasize a button by making it look like a link while maintaining button behavior

Button Sizes

Need smaller buttons? Add .btn-small or .btn-mini for additional sizes.

<p>
  <button class="btn btn-primary btn-small">Small Button</button>

  <button class="btn btn-tertiary btn-small">Small Button</button>
</p>

  <button class="btn btn-secondary btn-mini">Mini Button</button>
  <button class="btn btn-tertiary btn-mini">Mini Button</button>
</p>

How to Use

All icons require an aria-hidden="true" attribute with a unique class. The aria-hidden="true" prevents VoiceOver on OSX. To use, place the following code just about anywhere:

Note Since we are treating icons as fonts, we can color and size with ease and without degradation.

<span aria-hidden="true" class="icon-small-computergeneric>
<h3><span aria-hidden="true" class="icon-small-computergeneric green-text"><h3>
<h1><span aria-hidden="true" class="icon-large-computergeneric red-text"><h1>

Offsets

Sherpa includes built in margin offset classes that enable you to create additional space between elements.

class="" Description
bottom-offset-mini Adds 10px bottom margin
bottom-offset-small Adds 20px bottom margin
bottom-offset-medium Adds 40px bottom margin
bottom-offset-large Adds 60px bottom margin
