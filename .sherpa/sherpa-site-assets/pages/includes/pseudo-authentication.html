Pseudo Authentication

In a prototype you often have to show what a component might look like when a user is logged in vs not logged in. For this reason we have implemented a very simple system that tags the page as authenticated or not. This is used in conjunction with two css classes that can be used on any elements on the page.

  • authenticated-only: Only displays when the prototype is :"authenticated"
  • not-authenticated-only: Only displays when the prototype is NOT :"authenticated"

There are two ways to trigger a page's pseudo authentication:

  • Query string parameter: myPrototypePage.html?authenticated=true or myPrototypePage.html?authenticated=false
  • Javscript: Sherpa.publish("authenticated",true)

Both of these methods will set the page class and viewModel variable to enable dynamic display:

  • HTML Page classes: authenticated or not-authenticated
  • viewModel: $root.isAuthenticated

Markup:

Example:

Use to explicictly remove the element not used:						
<p data-bind="if: $root.isAuthenticated">I display beacuse this page IS authenticated</p>
<p data-bind="ifnot: $root.isAuthenticated">I display beacuse this page IS NOT authenticated</p>

Use when you still want to access the other element:
<p class="authenticated-only">I display beacuse this page IS authenticated</p>
<p class="not-authenticated-only">I display beacuse this page IS NOT authenticated</p>

Use when you want to conditionaly refer to different components:
<!-- ko if: $root.isAuthenticated -->
<div data-bind="component: { 
	name :'foobar', 
	type :'authenticated',
}"></div>
<!-- /ko -->
<!-- ko ifnot: $root.isAuthenticated -->
<div data-bind="component: { 
	name :'foobar', 
	type :'unauthenticated',
}"></div>
<!-- /ko -->

How it works:

The trigger mechanism adds either "authenticated" or "not-authenticated" to the html element.  There is css in Sherpa to work with this to provide classes that automatically:

html.authenticated .authenticated-only, 
html.not-authenticated .not-authenticated-only {
  display:block;
}


html.authenticated .not-authenticated-only, 
html.not-authenticated .authenticated-only {
  display:none;
}
