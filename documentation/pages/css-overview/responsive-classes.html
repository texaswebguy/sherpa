Responsive Classes

Dell 308 Style Guidelines:

The Bootstrap method is preffered for its simplicity - resize window to see responsive behavior.

NOTE: Gridset will be deprecated with Bootstrap 3.0 implementation.

Bootstrap Method

Desktop Only <div class="visible-desktop">View on desktop only... </div>
Hello Desktop!
Desktop went fishing!
Tablet Only <div class="visible-tablet">View on tablet only...</div>
Hello Tablet!
Tablet is out to lunch!
Phone Only <div class="visible-phone">View on phone only...</div>
Hello Phone!
Phone is out on a date!

Gridset Method

Desktop Only <div class="da-all ta-hide ma-hide">View on desktop only...</div>
Hello Desktop!
Desktop went fishing!
Tablet Only <div class="da-hide ta-all ma-hide">View on tablet only...</div>
Hello Tablet!
Tablet is out to lunch!
Phone Only <div class="da-hide ta-hide ma-all">View on phone only...</div>
Hello Phone!
Phone is out on a date!
