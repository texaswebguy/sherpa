Tabs

Overview

Tabs is a prevalent navigation system throughout the web. However, when we say "tabs" we mean any list of buttons, tabs, links, etc. that serve a purpose of switching panels or sub sections of content without leaving the page, not just something that looks like tabs.

Standard Tabs

Example (minimal with default data)

 

<div data-bind="component: { name :'tabs'}"></div>

 

Tabs below

 

<div data-bind="component: { 
    name :'tabs',
    options : {
		type : 'below'
	}
}"></div>

 

Tabs left

 

<div data-bind="component: { 
    name :'tabs',
    options : {
		type : 'below'
	}
}"></div>

 

Tabs right

 

<div data-bind="component: { 
    name :'tabs',
    options : {
		type : 'right'
	}
}"></div>

 

Color in tabs

You can color or stroke any tabs by addind the color to the tabbable class. For example:

 

<div data-bind="component: { 
    name :'tabs',
    options : {
    	color: 'blue',
    	id : 'switch_my_color'
	}
}"></div>

Beyond default data

All the examples above show the default sample data with three tabs. But what if you needed more tabs? What if you want actual content inside the tabs, etc. Follow instructions below to customize your tabs:

Specifying number of tabs (quantity)

 

<div data-bind="component: { 
    name :'tabs',
    options : {
    	color: 'blue',
	quantity: 4
}
}"></div>

 

Loading tabs with content (tab labels)

 

<div data-bind="component: { 
    name :'tabs',
    options : {
    	data: viewModel.core_config.default_tabs_data
}
}"></div>
