Alerts

There are two category of alerts available:

  • Page level alert: This is a rendering of the alert component on the top of each page which can be called at any time with javascript.
  • General alerts: Can be inserted anywhere on a page.

Page Level Alerts

See a sample page level alert

Markup

Trigger:

<a href="#" class="btn btn-tertiary page-level-alert-trigger" data-title="Sample Title for Alert" data-type="error" data-message="This is a sample message!">See a sample page level alert</a>

Attributes to pass information to page alert:
	data-title
	data-type
	data-message
	data-dismiss

data-alert_data can also be used instead to pass a data object from the view model.

If no data is passed then default data is displayed.

General Alerts

These can be inserted inside any container.

<div data-bind="component: { 
	name :'alerts', 
	options: {
		type: 'error',
		title: 'Error!',
		message: 'This is an error message.',
		dismiss:false
	}
}">
</div>

Options

  • type:
    • success
    • information
    • attention
    • error (default)
  • icon: accepts any icon class name
  • title: Can be any text. Localized by using msg function Ex: Sherpa.msg('textkey_label')
  • message: Can be any text. Localized by using msg function Ex: Sherpa.msg('textkey_label')
  • dismiss: true (default) / false

Examples

<div data-bind="component: { 
	name :'alerts', 
	options: {
	    type: 'success',
	    title: 'Success!',
	    message: 'This is a successful message.',
	    dismiss:false
	}
}">
</div>
<div data-bind="component: { 
	name :'alerts', 
	options: {
        type: 'information',
        title: 'Something you should know!',
        message: 'This is an informational message message.',
        dismiss:true
	}
}">
</div>
<div data-bind="component: { 
	name :'alerts', 
	options: {
	    type: 'attention',
	    title: 'Warning!',
	    message: 'This is an attention message.',
	    dismiss:false
	}
}">
</div>
<div data-bind="component: { 
	name :'alerts', 
	options: {
        type: 'error',
        title: 'Error!',
        message: 'This is an error message.',
        dismiss:false
	}
}">
</div>
