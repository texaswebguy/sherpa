Dell 308 Styleguide Rules:

          
NOTE: Dell 308 button class names deviated greatly from Bootstrap. They are now normalized using standard BS class names.

          
    
            
  • .btn-primary should be changed to .btn-success
    • 
            
  • .btn-secondary should be changed to .btn-primary
    • 
            
  • .btn-tertiary should be deprecated default .btn should be used instead.
    • 
            
  • .btn-info, .btn-warning, .btn-danger and .btn-inverse are not to be used and will display same as default.
    • 
          

          
 

          
You can check out the button builder at Bootsnip.com. Just beware that our icon classes are different.

Dell 308 Styleguide Rules:

          
No two popovers should display at the same time.  Default bootstrap behavior is to allow multiple popovers to display therefore they should be manually triggered to hide all other instances before displayed.

Javascript code to activate popovers

          
          
$('[data-toggle="popover"]').popover({trigger:'manual'});
  $('[data-toggle="popover"]').click(function(event){
    event.preventDefault();
    $('[data-toggle="popover"]').popover('destroy');
    $(this).popover('show');
  });
  $('[data-toggle="popover"][data-trigger="hover"]').mouseover(function(event){
    event.preventDefault();
    $('[data-toggle="popover"]').popover('destroy');
    $(this).popover('show');
  });
  $('[data-toggle="popover"][data-trigger="hover"]').mouseout(function(event){
    $(this).popover('destroy');
  });

  $('#custom-popover').click(function(event){
    event.preventDefault();
    $(this).popover('show');
    $('[data-dismiss="popover"]').click(function(event){
      event.preventDefault();
      $(this).parents('.popover').prev().popover('hide');
    });
  });

Responsive Classes

          

            
Dell 308 Style Guidelines:

            
The Bootstrap method is preffered for its simplicity - resize window to see responsive behavior.

            
NOTE: Gridset will be deprecated with Bootstrap 3.0 implementation.

          


          
Bootstrap Method


          
Desktop Only <div class="visible-desktop">...</div>
            
Hello Desktop!

            
Desktop went fishing!

          

          
Tablet Only <div class="visible-tablet">...</div>
            
Hello Tablet!

          
Tablet is out to lunch!

          

          
Phone Only <div class="visible-phone">...</div>
            
Hello Phone!

            
Phone is out on a date!

          

        

        

          
Gridset Method

        
        
        

          
Desktop Only <div class="da-all ta-hide ma-hide">...</div>

Tablet Only <div class="da-hide ta-all ma-hide">...</div>

          
Hello Tablet!

          
Tablet is out to lunch!

          
Phone Only <div class="da-hide ta-hide ma-all">...</div>

          
Hello Phone!
