Custom Configurations

The file assets/js/app.js is where your prototype application is. Here you can add custom configuration variables to the global data view model. These can be used to set up conditionals to display certain components or not.

All you have to do is add an element to the addToConfigs object. The sample below would be available in the view model (viewModel.config.sample_custom_config).

addToConfigs = {
  sample_custom_config : true
}
