Core Configuration

By editing assets/config/global_config.js you can change core settings. This javascript file contains the core of Sherpa such as paths, prototype default application, Javascript core libraries, CSS core files, default css and default locale.

This is what the Sherpa configuration object looks like once it is built on startup:
