Versatile:

The PowerVault NX3200 integrates with your existing Windows® environment, so there's no extra cost to deploy.

High-density:

Customize the NX3200 with up to 48TB capacity (SAS, NL-SAS and SATA drives available) in a 2U rack-mount chassis.

Easy to integrate:

Windows® Storage Server 2012 is preinstalled, so you can get up and running without having to build a file server.

Flexible:

The NX3200 supports file data as well as applications (block data), so you don't need to manage different types of storage.

Secure:

The NX3200 protects your data with RAID on enterprise-class hard drives with 24/7 duty cycles.

Easy to manage:

Manage the NX3200 from a remote office using the familiar Windows® Server interface or Dell’s iDRAC7.
